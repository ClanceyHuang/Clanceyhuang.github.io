Centos7 部署

在我看来CentOS7就是比6多了个SE，网上搜到的大部分资料都是直接关闭了SE，但并不认同。

  • 配置防火墙，开启 80 端口、3306 端口

CentOS 7.0默认使用的是firewall作为防火墙，这里改为iptables防火墙。

  • 先暂时关闭 firewall：
  • 安装 iptables 防火墙
  • 关闭 SELINUX
  • 安装 Apache

在客户端浏览器中打开服务器IP地址，会出现 Apache 欢迎页面，说明apache安装成功。

  • 安装 MariaDB

CentOS 7.0默认源中，已经使用MariaDB替代了MySQL数据库，如果要安装MySQL，替换对应的源即可。

  • 为 root 账户设置密码

回车，根据提示输入Y
输入2次密码，回车
根据提示一路输入Y
最后出现：Thanks for using MySQL!
MySql密码设置完成，重新启动 MySQL：

  • 安装 PHP 组件，使 PHP 支持 MariaDB
